# Installing `gtars`
While `gtars` is primarily a rust library, most users will interact with it through the Python package or command-line interface. The Python package provides bindings to the rust library, and the command-line interface provides a set of tools for working with genomic interval data.

## Installing for python
To install the Python package, use pip:

```bash
pip install gtars
```

## Installing the command-line interface
To install the command-line interface, use cargo:

```bash
cargo install --git=https://github.com/databio/gtars
```
